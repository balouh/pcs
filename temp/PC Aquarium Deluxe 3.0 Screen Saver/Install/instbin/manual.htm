1. Installation:
After inserting the CD into your computer's CD-Rom drive, installation should start automatically. If not, simply begin the process manually by double-clicking on the file 'setup.exe' from the CD. Once installation is under way, please follow the instructions on screen to complete the process. You can find the serial number / license code on the cover card / booklet. If you modify the installation path of PC Aquarium Deluxe 3.0, please modify the installation path of the bonus pack accordingly.

2 . System requirements:
Windows 98 (SE), Me, 2000 or XP, Pentium 450Mhz or higher, 64 MB RAM, 20MB disk space, 3D accelerator graphics card with 8MB Video RAM.

4. PC Aquarium Deluxe 3.0 Bonus Pack!
The bonus pack installs 4 new aquariums as well as 5 new fishes. These aquariums and fishes will appear after you restart PC Aquarium deluxe 3.0.

Personal backgrounds:
You can configure your own personal background by pressing CTRL-T. From this new window you can then import your own backgrounds / photos and save them separately. All common file formats are supported.

Transparent backgrounds:
PC Aquarium Deluxe supports transparent backgrounds. To import transparent backgrounds, it is necessary to use a specific file format called DDS (texture format). To create DDS files, you will need the NVIDIA DDS Plug-In (.8BF-format). Many graphic programs support this plug-in type (e.g. Paintshop, PhotoImpact, Photoshop etc.). For more information regarding plug-in installation as well as the plug-in itself, please consult your manual.

We recommend a maximum resolution of 1024x1024 pixel. If the resolution is
higher than the recommended, PC Aquarium Deluxe will ask you to use the file
anyway. If so the file size will be converted accordingly.
